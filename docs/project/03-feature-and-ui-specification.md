# 03 页面与交互

## 七页信息架构

1. 首页总览：数据、模型、GPU、API 和 SQLite 状态。
2. 数据中心：固定内置数据校验、预处理、质量证据和预览。
3. 用电分析：KPI、趋势、周期和分项结构。
4. 负荷预测：冻结模型、起点、设备、预测图、指标和 CSV。
5. 监测预警：96 点回放、三类预警、筛选和 CSV。
6. 智能建议：本地模板与一个主动 API 按钮。
7. 系统设置：只读环境、存储和 API 配置状态。

## 数据中心边界

页面只展示课程固定 CSV，不出现上传控件、字段映射、多数据集选择或外部数据源。

## 智能建议页

- 页面加载时先显示本地模板；
- 未配置 API 时不显示调用按钮；
- 配置完成后显示“调用大模型生成简短建议”；
- 点击前不发生网络请求；
- 页面可查看将发送的聚合 JSON；
- 成功时标记 API 生成；失败时显示模板回退；
- 大模型不参与数据、预测和预警计算。

## 已删除页面

“优化决策”从导航和验收范围移除。最终应用保持七页。
