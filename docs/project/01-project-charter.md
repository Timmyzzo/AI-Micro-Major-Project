# 01 项目范围

## 目标

使用课程固定家庭用电数据，完成可复现的数据校验、15 分钟聚合、历史分析、24 小时负荷预测、共形区间、历史回放和确定性预警，并通过一次可选大模型 API 调用生成简短中文建议。

## 必须保留

- 固定内置 CSV 的身份和质量校验；
- 缺失段识别、短缺失插值和长缺失保护；
- 固定月份 train/validation/test 切分；
- 历史 KPI、趋势、周期和分项分析；
- 朴素基线、Ridge、LSTM、PatchTST 公平比较；
- 90% 共形预测区间；
- 冻结模型推理、缓存和 CSV 导出；
- 历史回放、质量/规则/残差预警；
- 本地模板建议；
- 用户点击后的一次 Chat Completions 简短建议调用；
- 无 Key、断网或 API 失败时仍可演示核心功能。

## 明确删除

- 上传任意 CSV、字段映射和多数据集；
- 优化调度、峰谷电价、负荷转移和真实控制；
- 完整大模型报告系统、HTML 和报告数据库；
- REST、Docker、云端、多用户、通知和真实设备。

## 成功标准

1. 固定数据、分析、预测和预警可在本机复现。
2. 模型结果来自固定测试集，负结果不隐藏。
3. API 建议只发送聚合摘要，失败时回退本地模板。
4. 七页应用可启动，关键页面有自动测试。
5. 最终 Word/PDF 使用真实截图和实测指标。
