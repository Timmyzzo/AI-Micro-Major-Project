# 13 课程报告

本项目只准备课程要求的 Word/PDF 报告。

## 建议题目

**基于时序预测与确定性预警的家庭用电分析可视化系统**

## 章节

1. 引言
2. 数据集与预处理
3. 系统设计
4. 历史用电分析
5. 负荷预测模型与共形区间
6. 历史回放与预警
7. 简短智能建议功能
8. 测试与结果讨论
9. 问题与解决方法
10. 总结与展望
11. 参考文献
12. 附录：启动和使用说明

## 必须使用真实结果

- 260,640 条记录和实际缺失事实；
- 17,376 个 15 分钟点；
- M3 实际 KPI；
- 六模型真实测试指标；
- 复杂模型未超过最佳基线的讨论；
- 90% 区间实际覆盖；
- 三类预警实际示例；
- 七页最终截图；
- 测试、训练、推理和预警的实测结果。

## 不写入报告

- 已取消的上传、字段映射、优化决策、完整智能报告或云部署；
- 虚构的模型效果、费用节省或设备控制；
- API Key、本机隐私路径和个人敏感信息；

## 提交检查

- 封面、目录和页码正确；
- 图表有编号、单位和来源；
- Word/PDF 均可打开；
- 截图来自最终代码；
- 指标与仓库产物一致；
- 参考文献格式统一；
- 没有 Key、权重、缓存或个人隐私泄漏。
