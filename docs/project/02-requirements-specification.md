# 02 需求规格

## P0 功能需求

### DATA 固定课程数据

- FR-DATA-001：只使用 `docs/household_power_consumption.csv`。
- FR-DATA-002：校验字段、时间、重复、缺失、采样间隔和明显无效值。
- FR-DATA-003：保留原始文件只读，生成分钟和 15 分钟 Parquet、manifest 与稳定 ID。
- FR-DATA-004：短缺失可插值，长缺失不得跨段填充。

不提供上传、字段映射、多数据集或数据版本切换。

### MON 历史分析

- FR-MON-001：展示区间覆盖率、累计电量、平均、峰值和最低功率。
- FR-MON-002：展示确定性历史趋势和周期规律。
- FR-MON-003：展示三个分项和未分项能量证据。

### FCST 负荷预测

- FR-FCST-001：使用 672 个 15 分钟历史点预测未来 96 点。
- FR-FCST-002：比较朴素基线、Ridge、LSTM 和 PatchTST。
- FR-FCST-003：模型选择只使用验证集，测试集只做最终评估。
- FR-FCST-004：输出点预测、90% 区间、真实值和区间外标记。
- FR-FCST-005：支持冻结模型加载、离线缓存和 CSV 导出。

### ALERT 回放与预警

- FR-ALERT-001：对一个 96 点历史回测片段继续、暂停、单步和重置。
- FR-ALERT-002：把质量问题转换为质量预警。
- FR-ALERT-003：使用训练段稳健阈值生成负荷和变化规则预警。
- FR-ALERT-004：真实值落在预测区间外时生成残差预警。
- FR-ALERT-005：等级由确定性规则计算并支持 CSV 导出。

不实现确认、忽略、备注或预警数据库。

### ADVICE 简短建议

- FR-ADVICE-001：本地模板根据聚合结果生成不超过三条建议。
- FR-ADVICE-002：用户配置并主动点击后，可调用一次 Chat Completions。
- FR-ADVICE-003：发送内容只包含数据身份、时间范围、覆盖率、电量、平均/峰值负荷、默认模型及测试 MAE和限制。
- FR-ADVICE-004：不得发送原始 CSV、完整时序、Key、权重或本地路径。
- FR-ADVICE-005：API 未配置、失败或返回空内容时回退本地模板。

不实现完整报告、Responses、自动探测、结构化输出、HTML、历史记录或多轮对话。

## 非功能需求

- 页面刷新不训练模型、不自动调用 API。
- 原始数据只读，Key 不写入 YAML、日志、SQLite、导出或截图。
- 相同数据和配置产生相同分析、预测缓存与预警 ID。
- 颜色不是状态的唯一表达方式。
- Ruff、格式、mypy、pytest 和 pre-commit 必须通过。
- 所有结论标明“课程演示，不作为电气安全或调度依据”。
