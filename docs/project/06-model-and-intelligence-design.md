# 06 模型、预警与智能建议

## 预测

- 输入窗口：672 个 15 分钟点；
- 输出窗口：96 点，即未来 24 小时；
- 基线：Last Value、前一日同刻、前一周同刻、Ridge；
- 对照模型：小型 LSTM；
- 主实验模型：小型单变量 PatchTST；
- 缩放器只在训练集拟合；
- 选模只看验证集；
- 90% 区间由验证残差按预测步长校准。

正式结果中，前一日同刻测试 MAE 为 0.6463 kW，是默认模型；PatchTST 测试 MAE 为 0.6622 kW。项目保留复杂模型未胜出的事实。

## 预警

- 数据质量预警来自 manifest；
- 规则预警阈值只从训练段 median、MAD 和分位数推导；
- 残差预警由真实值是否落在共形区间外决定；
- 大模型不能改变预警等级。

## 简短建议

本地模板是默认能力。可选 API 只做一次 Chat Completions 文本生成：

1. 程序先计算聚合摘要；
2. 用户点击按钮；
3. API 最多输出三条中文建议；
4. API 不得补造数值或声称电气故障；
5. 失败或空响应时回退本地模板。

不实现完整报告、结构化输出、Responses、自动探测、多轮对话、优化建议或报告历史。
