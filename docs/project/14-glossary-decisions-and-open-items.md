# 14 术语与决策

## 术语

| 术语 | 含义 |
| --- | --- |
| 固定内置数据 | 仓库课程 CSV，唯一数据源 |
| dataset_id | 原始数据稳定身份 |
| preprocess_id | 预处理规则和数据身份 |
| 672→96 | 7 天历史预测未来 24 小时 |
| 共形区间 | 用验证残差校准的 90% 预测区间 |
| 残差预警 | 真实值落在预测区间外的确定性预警 |
| 本地模板 | 无 API 时生成的简短建议 |
| API 建议 | 用户主动触发的一次 Chat Completions 文本生成 |

## 有效决策

1. Streamlit 单体，不增加 REST 服务。
2. 只使用固定课程 CSV，不支持上传和映射。
3. 原始数据只读，时序存 Parquet，SQLite 只存轻量元数据。
4. 固定月份切分，训练集拟合缩放器，验证集选模。
5. PatchTST 是课程实验模型，但默认模型由真实测试结果决定。
6. 预警等级完全由程序规则决定。
7. 删除优化决策和完整智能报告。
8. 大模型只保留一个短建议入口，使用 Chat Completions。
9. API 只接收聚合摘要，失败回退本地模板。
