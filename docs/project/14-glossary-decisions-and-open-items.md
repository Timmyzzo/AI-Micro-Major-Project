# 14 术语与决策

## 术语

| 术语 | 含义 |
| --- | --- |
| 当前案例数据 | 仓库内已经适配、训练和验收的课程 CSV；不是课程允许使用的唯一数据来源 |
| dataset_id | 原始数据稳定身份 |
| preprocess_id | 预处理规则和数据身份 |
| 672→96 | 7 天历史预测未来 24 小时 |
| 共形区间 | 用验证残差校准的 90% 预测区间 |
| 残差预警 | 真实值落在预测区间外的确定性预警 |
| API 建议 | 用户主动触发的一次 Chat Completions 文本生成 |
| API 连接测试 | 首页向当前模型发送一条最小测试消息并显示回复与耗时 |
| 展示默认 | 当前注册表中按测试 MAE 标记、用于页面默认选择的模型，不等同于验证集独立选模结果 |
| 回归准确度 | 用 MAE、RMSE、WAPE、sMAPE、R²、区间覆盖率或明确定义的容差命中率表达 |

## 有效决策

1. Streamlit 单体，不增加 REST 服务。
2. 当前实现只读取已经适配的课程 CSV，不支持上传和映射；开发阶段允许更换公开电力数据，但必须更新契约并重建产物。
3. 原始数据只读，时序存 Parquet，SQLite 只存轻量元数据。
4. 固定月份切分，训练集拟合权重与缩放器，验证集用于早停和内部选参，冻结候选在测试集上统一展示。
5. PatchTST 是课程实验模型；当前页面展示默认由测试 MAE 最低候选决定，文档不得把它写成验证集独立选模结果。
6. 预警等级完全由程序规则决定。
7. 删除优化决策和完整智能报告。
8. 大模型只保留一个短建议入口，使用 Chat Completions。
9. API 只接收聚合摘要，成功显示模型回复，失败显示明确状态。
