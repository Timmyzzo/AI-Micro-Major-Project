## 主要风险

| 风险 | 处理 |
| --- | --- |
| 单户数据泛化有限 | 报告明确限制 |
| 复杂模型未超过基线 | 保留真实负结果 |
| API 未配置、网络中断或模型通道异常 | 系统核心数据、分析、预测和预警仍可运行，页面显示明确连接状态 |
| API 编造数值 | 提示只允许解释聚合摘要；建议不用于计算 |
| 演示网络不可用 | 核心流程完全离线 |
| 范围再次膨胀 | 新功能默认拒绝，优先完成报告和截图 |
