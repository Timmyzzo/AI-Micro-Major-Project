## API 验收

自动测试不得访问网络，使用 mock 验证：

- 只调用一次 `chat.completions.create`；
- 请求包含聚合 JSON；
- 不包含原始 CSV 或完整时序；
- 空响应和异常回退本地模板；
- 未配置 Key 时页面不显示调用按钮；
- 页面加载不发起请求。

真实 API 调用是可选人工验收，不阻塞课程核心功能。
